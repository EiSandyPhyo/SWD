Color

  1. Solid Color
    • Color Code
      1. name
        • black, red, green, blue, white
      2. hex
        • #123123
        • total 6 char & number
        • 0,1,2,...9,a,b,...f
        • rrggbb
      3. rgb
        • rgb(123,123,123)
        • 0-255
        • rgb(r,g,b)
      4. hsl
        • hsl(hue, situation, lightness)
        • H -> 0-360 => 0-red, 120-green, 240-blue
        • S -> % => 0%-gray, 100%-full color
        • L -> % => 0%-black 100%-white
  2. Transparent Color
    • Color Code
      1. hex
        • total 8
        • #rrggbb[00-aa]
      2. rgb
        • rgba(r,g,b,a)
        • a[0 - 0.5 - 1]

Art is an expression of our thoughts, emotions, intuitions, and desires, but it is even more personal than that: it's about sharing the way we experience the world, which for many is an extension of personality. It is the communication of intimate concepts that cannot be faithfully portrayed by words alone. And because words alone are not enough, we must find some other vehicle to carry our intent.
